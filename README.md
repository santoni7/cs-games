# C# Games collection
Written for educational purposes and fun.

## SpaceShooter
A clone of famous *Asteroids* game, with infinite level and increasing difficulty. 
Made with:
- Visual Studio 2015
- .net 4.5
- MonoGame

## StickRun
A simple C# platformer game.  Collect diamonds and avoid small stick-men. 
Made with:
- Visual Studio 2010
- .net 4.0
- XNA 4.0
